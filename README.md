# Game FAT 3 (gfat3)

This is a utility that allows you to store your game data in one massive file.

## FILES:

```gfat3.exe``` is used to create the repository file.

These objects are used to access the repository:

```
	sc/c_error.h 
	sc/c_filelib.h
	sc/c_gfat.h
```
